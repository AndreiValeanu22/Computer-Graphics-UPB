# Tema 3 — Firefly Forest

Modul pentru **gfx-framework** (`m1::Tema3`) — temă **suplimentară**.

## Integrare

La fel ca la Tema 2, dar cu `m1::Tema3` în `main.cpp`.

## Funcționalități implementate

- Teren, copaci cu frunziș pe nivele, turn de observație
- Licurici emisivi, ceață, light cookies
- Shadere custom cu animație de vânt

## Texturi

Incluse în `textures/`: `bark.jpg`, `leaves.jpg`, `wood_planks.png`, `grass.jpeg`, `wood.jpg`.
Modelele `cone.obj` și `cylinder.obj` trebuie adăugate local (sunt furnizate de framework).
